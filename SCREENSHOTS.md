# Screenshots Guide

To add screenshots to the README, take the following screenshots and save them in a `screenshots/` folder:

## Required Screenshots

### 1. `popup.png`
- Open the extension popup
- Show the clean launcher interface
- Capture: "Start Recording" button and instructions

### 2. `recorder-window.png`
- Open the recorder window
- Show the professional gradient UI
- Capture: REC/STOP buttons, status badge, and step-by-step instructions

### 3. `recording-in-progress.png`
- Start a recording
- Show the "Recording..." status with animation
- Capture: The orange pulsing status badge

### 4. `taskbar-stop.png`
- Show the minimized recorder in the Windows taskbar
- Highlight where to click to stop recording
- Capture: Taskbar with recorder window icon

## Tips for Good Screenshots

- Use 1920x1080 resolution
- Capture with clean background
- Ensure text is readable
- Use PNG format for quality
- Keep file sizes under 500KB each
